CheckSum Tool

CheckSum Tool is an utility program for calculating and verifying checksums for files. The tool is in development and new features will be added frequently in new releases.

Features

  • Cheksum types: CRC32, MD5 and SHA-1
  • File types: flv, md5 and sha1
  • Simple and fast to use
  • Written in .Net (C#)

Go to project pages at SourceForge.net.

Downloads

CheckSum Tool downloads are in three categories:

  • Alpha releases are testing releases that are not meant for production use.
  • Beta releases are preview releases for stable releases.
  • Stable releases are for production use.
